Panduan Cepat untuk Mengambil Screenshot Sistem

Installation terlalu lama? Gunakan metode manual yang lebih cepat di bawah!
Metode Tercepat (2-3 menit)
1Buka Browser

Buka Chrome/Firefox dalam mode incognito:

Ctrl + Shift + N (Chrome) atau Ctrl + Shift + P (Firefox)
2Atur Ukuran Desktop

Tekan F12 → Klik ikon device → Pilih "Responsive" → Set ke 1920 x 1080

3Screenshot Halaman Desktop
http://127.0.0.1:8000/login Login Page
http://127.0.0.1:8000/superadmin/dashboard Dashboard
http://127.0.0.1:8000/superadmin/users User Management
http://127.0.0.1:8000/mobile-test.html Mobile Test

Gunakan Ctrl + Shift + S atau klik kanan → "Save screenshot"

4Screenshot Mobile

Ubah ukuran ke 375 x 812 (iPhone 12 Pro) dan screenshot ulang semua halaman

5Simpan File

Simpan dengan nama:

  • 01-login-desktop.jpg
  • 02-dashboard-desktop.jpg
  • 03-users-desktop.jpg
  • 04-mobile-test-desktop.jpg
  • 05-login-mobile.jpg (dan seterusnya)

Pro Tips:
  • Gunakan incognito mode untuk menghindari cache
  • Screenshot full page untuk menampilkan seluruh konten
  • Simpan dengan kualitas tinggi (JPEG 90-100%)
  • Ambil screenshot mobile dan desktop untuk perbandingan
  • Beri nama file yang jelas untuk mudah diorganisir
